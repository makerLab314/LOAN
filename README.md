## Preview

Willkommen be <em>iOAN, dem System zur Geräteausleihe für wissenschaftliche und medienpädagogische Einrichtungen</em> von Vincent Dusanek. Das System wurde läuft auf einer klassischen LNMP-Struktur: 
<ul>
    <li>Ubuntu 24.04</li>
    <li><Nginx/li>
    <li>MariaDB</li>
    <li>PHP 8.3</li>
</ul>

Um Zugang zu einem Testaccount unter <a href="http://loan.vdus.de" target="_blank">http://loan.vdus.de</a> zu erhalten, kontaktieren Sie mich bitte unter vincent.dusanek[at]gmail.com.
